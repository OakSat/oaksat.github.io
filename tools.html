"Ethical Demos" Project - Teaching Demos

As a student mentor, I’ve had the chance to guide curious young minds into the world of cybersecurity — helping them grasp key concepts, real-world scenarios, and essential industry terminology.

The demos below are hands-on simulations of common cyber attacks, created strictly for educational use.
Each one follows ethical guidelines and aims to make security concepts more accessible and engaging.

Open redirect demo

A simple example showing how attackers can trick users into visiting malicious sites by abusing insecure URL redirections. Check it out on GitHub

Dictionary Attack Demo

A demonstration of how attackers can break weak passwords by rapidly trying common words from a predefined list. Check it out on GitHub
